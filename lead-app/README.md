# AI Lead Classification Tool

This project is a simple lead qualification system that classifies leads into:
- Decision Makers
- Individual Contributors

## Features
- CSV upload
- Automatic classification
- Confidence scoring
- High-value lead filtering
- Downloadable results

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Use Case
Helps sales teams identify high-value prospects quickly.
