# 🚀 Hacker News Scraper & Analyzer

This project is an asynchronous web scraping tool built with Playwright that extracts and analyzes data from Hacker News.

## 🔹 Features

- Scrapes multiple pages from Hacker News
- Extracts:
  - Title
  - Author
  - Points
  - Comments
  - URL
- Categorizes posts into:
  - Trending (100+ points)
  - Hot Discussion (50+ comments)
  - Normal
- Sorts posts by popularity
- Exports clean CSV file

## ⚙️ Tech Stack

- Python
- Playwright (async scraping)
- Pandas
